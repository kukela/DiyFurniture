## 1. 简介
这是一个Diy家具的项目，用尽可能低的成本实现好用的功能。

**设计工具：**
目前我使用的是免费的2D设计软件[Luncay](https://igoutu.cn/lunacy)，这个软件是跨平台的。[Luncay英文官网](https://icons8.com/lunacy)
