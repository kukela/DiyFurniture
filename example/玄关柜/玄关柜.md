## 1. 简介

通过几个可定制模块化柜子按需实现需要的功能，有拆卸简单，美观方便等特点，兼容10\~35cm深度。

## 2. 设计指南

示例设计文件[example\/玄关柜\/demo01.sketch](https://gitee.com/kukela/diy-furniture/tree/master/example/玄关柜/demo01.sketch)

**[设计大纲](https://gitee.com/kukela/diy-furniture/tree/master/doc/设计大纲.md)**

1. 一般情况下，玄关底部留15\~16cm的空挡，台面距离地面90\~110cm，中间镂空35\~50cm，顶高一般180cm以上（按照手能碰到的地方为宜）。

	[示例图]

2. 玄关柜应当优先保证收纳鞋子的数量，并且保证一定数量的鞋子可以方便拿取，可以参考《》中说的数量。

	[示例图]

3. 一双鞋子宽度一般是20\~24cm，鞋柜宽度最好按照24cm的倍数设计，常见鞋子尺寸参考《》

	[示例图]

4. 鞋柜深度30\~35cm适合平放的隔板，18\~30cm适合斜放的隔板，10\~18cm适合鞋子横放的隔板。

	[示例图]

5. 玄关柜应当不影响人正常通行，应当保证通道宽度不小于80cm，超过120cm则有些浪费空间。

	[示例图]

6. 在做台面的地方，建议用两根横杆支撑，能起到稳定台面物体的作用。

	[示例图]

7. 衣柜宽度一般53cm\~62cm按照自己需求来定义。

8. 一般大书包尺寸是 长32cm，宽16cm，高42cm，包包柜可以参考这个尺寸设计。

### 3.2. 雨衣柜

用于临时存放大衣、晾干雨衣、雨伞、衣服、手套等物件。有排水设计，柜子底下可以存放靴子，晾干雨鞋套。

| - | - | - |

**BOM清单：**
| 名称 | 数量 | 价格（￥） | 商品大纲代码 |
| - | - | - | - |
| | | | |

**组装步骤**

1. 前置参考：

	[铝型材](https://gitee.com/kukela/diy-furniture/tree/master/doc/DesignGuide/铝型材.md) -> 2. 指南 -> 全部条目。
	
	[EVA薄膜](https://gitee.com/kukela/diy-furniture/tree/master/doc/DesignGuide/EVA薄膜.md) -> 2. 指南 -> 全部条目。

2. 组装底柜框架。

	[示例图]

3. 安装隔层横杆，最上面的隔层需要做排水面，所以不装内横杆。

	[示例图]

4. 裁剪隔层EVA防滑垫到如下图所示，并打孔。
	
	[示例图]

5. 裁剪排水面EVA防滑垫到如下图所示，并打孔，注意安装的时候防滑垫是光滑面朝上的，需要注意裁剪左右关系。

	[示例图]

6. 安装隔层和排水面EVA防滑垫，排水面后面的对角需要向内折一下。

	[示例图]
	
7. 测试排水面是否能正排水。

	[示例图]

8. 组装高柜框架，顶部台面可以装。

	[示例图]

9. 裁剪高柜顶部台面EVA防滑垫，然后直接放上去。

	[示例图]

10. 底柜和高柜合体。

	[示例图]

11. 裁剪高柜和底柜的防水内胆，底柜防水内胆应超过防水台面5cm以上，推荐用0.3mm的EVA薄膜。

	[示例图]

12. 安装防水内胆。

	[示例图]

13. 在柜体内部两侧合适的位置用小刀划两个口子，用来挂雨伞。

	参考：

	[杂物挂钩](https://gitee.com/kukela/diy-furniture/tree/master/doc/DesignGuide/杂物挂钩.md)  -> 方孔S型挂钩、螺丝孔挂钩

	[示例图]

14. 在柜体支柱上安装挂绳，推荐上下两个，因为内胆挡住了铝型材槽，需要先在内胆上画上记号，然后打孔。

	参考：

	[挂绳](https://gitee.com/kukela/diy-furniture/tree/master/doc/DesignGuide/挂绳.md)  ->  铝型材钢丝挂绳

	[示例图]

15. 安装其他配件。

	参考：

	[杂物挂钩](https://gitee.com/kukela/diy-furniture/tree/master/doc/DesignGuide/杂物挂钩.md)  -> 方管S型挂钩、螺丝孔挂钩

	[夹子](https://gitee.com/kukela/diy-furniture/tree/master/doc/DesignGuide/夹子.md)  -> 横杆挂夹
